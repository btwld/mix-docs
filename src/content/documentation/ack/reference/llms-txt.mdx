# AI & llms.txt

Ack publishes a compact, machine-readable documentation index at [`/ack/llms.txt`](/ack/llms.txt). It is generated alongside the library documentation and links AI agents to the canonical guides and API surface.

The repository root [`llms.txt`](https://github.com/btwld/ack/blob/main/llms.txt) remains the canonical source. The website serves the same content as plain text so tools can discover it without parsing HTML.
